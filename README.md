# 💱 Currency Converter

A simple and responsive **Currency Converter** built using **HTML, CSS, and JavaScript** that allows users to convert one currency to another using real-time exchange rates.

---

## 🚀 Features

* Convert between different currencies
* Real-time exchange rates using an API
* Country flags displayed for selected currencies
* Simple and responsive UI
* Easy to use dropdown currency selection

---

## 🛠️ Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and layout
* **JavaScript** – Logic and API integration
* **Exchange Rate API** – For live currency conversion

## ⚙️ How It Works

1. User selects **From Currency**
2. User selects **To Currency**
3. Enters the **amount**
4. Clicks the **Convert** button
5. The app fetches the **latest exchange rate** from the API and displays the converted amount.
